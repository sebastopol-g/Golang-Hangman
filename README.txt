This project is a simple Hangman game.
It's a simple introduction to the Go language.

Game's rules :
You have 8 chances to find the hidden word.
You have to make a guess (enter a letter to see if the word contains it)
If you make a bad guess, you lose a life.
If you enter the same guess as before, the game continues and you don't lose a life.
If you find all the letters of the hidden word, you win and the game stops; Otherwise you lose.
